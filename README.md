## 1、喜马拉雅app极速版cookie获取方法
+ 安装极速版app并注册登陆账号
+ 小黄鸟或则Fiddler抓包
+ 找到 m.ximalaya.com 复制其cookie值
## 2、喜马拉雅cookie设置方法
+ 替换 himalaya_cookie.json文件  cookie: XXX" 为自己的Cookie值

## 3、注意事项
+ 只要cookie值其他的不要
